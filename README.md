# tugas_python_ai_b10

Repository ini berisi kumpulan tugas infinite learning Python dan AI.

---

## Tugas 4 – Struktur Data Python

Tugas ini membahas berbagai struktur data bawaan Python.
List digunakan untuk menyimpan kumpulan data yang bisa diubah,
lengkap dengan operasi seperti append, insert, extend, pop, dan remove.
Tuple dipakai untuk data yang tidak perlu diubah, dengan fitur unpacking
yang memudahkan pengambilan nilai. Set berguna saat kita butuh data unik
dan operasi himpunan seperti union, intersection, dan difference.
Dictionary menyimpan data dalam format key-value yang mudah diakses dan diubah.
Selain itu, tugas ini juga mencakup struktur bersarang, list/dict/set comprehension,
dan pengecekan keanggotaan menggunakan operator in.

---

## Tugas 5 – Fungsi dan OOP Dasar

Tugas ini memperkenalkan cara membuat fungsi di Python dengan parameter,
nilai default, dan type hint. Fungsi yang dibuat antara lain greet(), tambah(),
dan rata_rata() yang dipakai ulang di dalam kelas. Konsep OOP diterapkan
melalui kelas Student yang memiliki atribut nama, nim, dan daftar nilai,
serta metode untuk menambah nilai, menghitung rata-rata, dan menentukan
status kelulusan. Method __str__ digunakan untuk menampilkan informasi
objek secara ringkas dan mudah dibaca.

---

## Tugas 6 – NumPy, Pandas, dan OOP Lanjutan

Tugas ini menggunakan library NumPy untuk operasi statistik pada array nilai ujian,
seperti menghitung rata-rata, median, standar deviasi, nilai minimum dan maksimum.
Library Pandas dipakai untuk membuat DataFrame berisi data mahasiswa,
menambahkan kolom status kelulusan secara otomatis berdasarkan nilai,
dan menampilkan data secara terstruktur. Kelas GradeBook dibuat untuk
membungkus DataFrame dan menyediakan metode average(), pass_rate(),
dan save_summary() yang menyimpan ringkasan ke file .txt.
Tugas ini menggabungkan konsep OOP dengan pengolahan data nyata menggunakan library populer.
